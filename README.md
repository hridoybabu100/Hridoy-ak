# Hridoy Ak - Premium Portfolio

A cinematic, high-performance personal portfolio built with **Next.js 14**, **Framer Motion**, **GSAP**, and **Lenis** smooth scrolling. Designed with futuristic aesthetics, glassmorphism, and buttery-smooth animations.

![Hero Preview](https://via.placeholder.com/1200x600/0a0a0a/ffffff?text=Hridoy+Ak+Portfolio)

## ✨ Features

- **Cinematic Animations** — Powered by GSAP + Framer Motion
- **Ultra-Smooth Scrolling** — Lenis + Custom Scrollbar
- **Magnetic Interactions** — Premium hover effects
- **Custom Cursor** — Futuristic glowing cursor
- **Glassmorphism Design** — Modern dark UI with neon accents
- **Fully Responsive** — Mobile-first approach
- **Loading Animation** — Progress bar with confetti on form submit
- **Performance Optimized** — Next.js App Router + Image Optimization

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/hridoybabu100/hridoy-ak-portfolio.git

# Navigate to project directory
cd hridoy-ak-portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [https://hridoy-ak.vercel.app/](http://localhost:3000) to view the portfolio.
